Split the pipeline state machine — the phase's actual target.

is not the largest, but it absorbed **3 of Phase 18's 7 plans** (18-04, 18-05,

`pipeline_outcomes.rs`, and `pipeline_gate.rs`.

Purpose: this is the highest-value cut in the phase, and it is deliberately
scheduled in wave 4 — early enough that if D-12's partial-completion outcome is
invoked, the pipeline win has already landed, and late enough that the
extraction procedure was proven on two smaller clusters in 19-07 first.
Output: three sibling files and a `main.rs` roughly 1,040 production lines and
~1,700 test lines lighter.
