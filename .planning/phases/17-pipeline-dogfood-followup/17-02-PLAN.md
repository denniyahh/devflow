<objective>
Add the workspace's first build script so the devflow binary can identify the source it was built from. Shell out to git at compile time and emit build provenance (commit, dirty flag, timestamp) as compile-time environment variables, degrading gracefully when git metadata is unavailable.

Purpose: The Phase 16 incident consumed an entire spike phase reading false evidence from a stale Homebrew-symlinked binary. Embedded provenance is the root-cause fix — it makes a stale self-dogfood build detectable (consumed by Plan 05). This plan produces only the compile-time half; runtime consumption and staleness detection are Plan 05.
Output: crates/devflow-cli/build.rs and an integration test asserting the env vars resolve.
</objective>
