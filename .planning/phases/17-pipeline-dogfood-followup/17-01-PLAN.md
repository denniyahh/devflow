<objective>
Establish the typed-outcome taxonomy and the fail-closed policy table that the rest of Phase 17 depends on. Add the two missing agent outcomes (ResourceKilled exit 137, AgentUnavailable exit 127), classify them in Layer 2, add a structured "which layer decided" field to AgentResult, and extract the outcome->action decision into a pure, exhaustively-matched function in devflow-core so the never-advance guarantee is compiler-enforced. Add the separate infrastructure-failure counter State field.

Purpose: A hardcoded exhaustive policy table is the mechanism that prevents the D-01 regression class (a new/unhandled outcome silently advancing). This plan is the foundation Plan 03 (cascade rework) and Plan 04 (advance dispatch) build on.
Output: New AgentStatus variants with correct wire names, Layer 2 exit-code classification, outcome_policy module, State.infra_failures, mode ceiling constant.
</objective>
